Query relationships#

This query allows you to retrieve the relationships with the optional creator address.

CLI

desmos query profiles relationships [[address]]
# Example# desmos query relationships# desmos query relationships desmos13p5pamrljhza3fp4es5m3llgmnde5fzcpq6nud
