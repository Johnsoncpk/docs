Validators#

Requires Desmos executable

To join the public testnet you must have desmos installed.

To become a testnet validator, the mainnet instructions apply:

  1. Create a full node.
  2. Become a validator

The only difference is the Desmos software version and genesis file. See the testnet repo for information on testnets, including the correct version of Desmos to use and details about the genesis file.
